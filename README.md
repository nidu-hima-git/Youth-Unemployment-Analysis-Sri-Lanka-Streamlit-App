# Youth-Unemployment-Analysis-Sri-Lanka-Streamlit-App
Interactive Streamlit dashboard developed for analysing youth unemployment patterns in Sri Lanka using Labour Force Survey (LFS) 2023 data.

## Features
- Data exploration and preprocessing summary
- Interactive visualisations
- Logistic Regression model
- Decision Tree model
- Hypothesis testing
- User-friendly dashboard interface

## Tools & Technologies
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- Seaborn

## Dataset
Sri Lanka Labour Force Survey (LFS) 2023  
Department of Census and Statistics (DCS), Sri Lanka
