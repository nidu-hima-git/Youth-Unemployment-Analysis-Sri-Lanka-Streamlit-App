# Youth Unemployment Analysis Dashboard – Sri Lanka

This project presents an interactive dashboard developed using Streamlit to analyze youth unemployment patterns in Sri Lanka using the Labour Force Survey (LFS) 2023 dataset.

The dashboard was developed as part of the ST3011 Statistical Modelling group project and extends the group analysis by providing an interactive and user-friendly platform for exploring unemployment trends among Sri Lankan youth.

---

## Project Objective

The main objective of this dashboard is to provide an interactive platform to explore youth unemployment patterns based on:

- Gender
- Age
- Educational Attainment
- Residential Sector
- Province / District
- Employment Status

The dashboard enables users to dynamically analyze relationships and patterns beyond the static visualizations included in the final report.

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Features of the Dashboard

✅ Interactive unemployment visualizations  
✅ Gender-based unemployment analysis  
✅ Sector-wise unemployment comparison  
✅ Age distribution analysis  
✅ Educational attainment analysis  
✅ Province and district-level comparisons  
✅ Logistic regression model insights  
✅ Weighted logistic regression for imbalanced data  
✅ Interactive filters and charts  

---

## Dataset
Sri Lanka Labour Force Survey (LFS) 2023  
Department of Census and Statistics (DCS), Sri Lanka

---

## How to Run the App
- Install required libraries:
   - pip install streamlit pandas numpy matplotlib seaborn scikit-learn
- Run the Streamlit app:
   - streamlit run app.py
